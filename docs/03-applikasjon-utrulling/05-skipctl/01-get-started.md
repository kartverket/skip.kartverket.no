## Installasjon

Last ned [siste versjon](https://github.com/kartverket/skipctl/releases) eller bruk det medfølgende Docker-imaget (hovedsakelig for å kjøre en server).

### Bruk Homebrew (mac)

```shell
brew tap kartverket/taps && \
brew install skipctl
```
